# Bharath - Backend & Automation

**Role:** Backend & Automation Developer

## Responsibilities
1.  **Automation Engine**:
    *   **App Control**: Opening and managing Windows applications (WhatsApp, Instagram, etc.).
    *   **Input Simulation**: Simulating keyboard and mouse events.
    *   **Screen Vision**: Using OCR and computer vision to read screen content.
    *   **Voice Module**: Handling voice commands and text-to-speech.
2.  **Backend API**: The core server processing requests from the frontend and AI core.
3.  **MT5 Bridge**: Managing the secure connection to the MetaTrader 5 bot.

## Key Folders
*   `automation_engine/`: Core scripts for OS interaction.
*   `social_integrations/`: Specific logic for social platforms.
*   `mt5_bridge/`: Connection logic for trading.
