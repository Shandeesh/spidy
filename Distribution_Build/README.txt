=== SPIDY AI DISTRIBUTION INSTRUCTIONS ===

TO DISTRIBUTE:
1. Copy this ENTIRE folder "Spidy_Dist" to the client's "D:\" drive (or C:\).
2. On the client's machine, Open the folder.
3. Right-Click "Setup_Spidy.ps1" -> "Run with PowerShell".

INSTALLATION PROCESS:
1. The installer will show a "Hardware ID".
2. The Client must send you this ID.
3. YOU generate an Activation Key using "ADMIN_TOOLS\Generate_Key.ps1" on YOUR machine.
4. Give the key to the Client.
5. Client enters the key.
6. Installation completes automatically.

FILES:
- Setup_Spidy.ps1: The Main Installer.
- dist_server.py: The Application Server.
- frontend\: The App Interface.
- spidy_lib\: The Brain & Trading Logic.
